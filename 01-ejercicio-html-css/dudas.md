# Dudas que me surgen según realizo el ejercicio.

## background-image o <img>

¿Qué es mejor para un fondo de una sección con una imagen (como en el hero)? ¿Poner un img en HTML o usar la propiedad background en CSS?

RESPUESTA: Hola genio! Excelente pregunta :)
Todo depende si la imagen es decorativa, o tiene valor para el contenido, te doy un ejemplo de la vida real:
Si vamos por la calle y encontramos un cartel de `se busca perro perdido`, entendemos que una foto del perro es un contenido valioso para el propósito del mensaje, entonces, en este caso usamos `<img />`. Ahora, en ese mismo cartel, tenemos una imagen de una patita en el lateral, como forma de decoración, entonces, en este caso usamos `background-image`.

Resumen: Cuando la imagen tiene valor para el contenido, usamos `<img />`, cuando es decorativa, usamos `background-image`.
